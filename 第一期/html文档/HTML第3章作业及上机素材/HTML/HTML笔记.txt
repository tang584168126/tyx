网页基本结构组成：

<html>
	
	<head>
		<title>S1T182班</title>
	</head>
	
	<body>
		1111111111111
	</body>
</html>

h1 ~ h6 为 标题标签
<h1>	
<h2>
<h3>
<h4>
<h5>
<h6>

<h1></h1>

<p> 	段落标签


<br/>	换行标签（只能这样写）


<hr/>	水平线标签（只能这样写）

<img>	图像标签
	src	要显示的图像路径
		路径：
			绝对路径
				c:\a\1.jpg
			相对路径
				文档相对路径
				1.jpg
				Images/1.jpg
				
				站点根目录相对路径
				/Images/1.jpg
	alt		图片的替代文字(也可用于鼠标悬停提示)
	title	鼠标悬停提示文字
	width	图片宽(如果没有设置，将使用图片原始宽度)
	height	图片高(如果没有设置，将使用图片原始高度)
	
<a/>	超链接标签(跳转到对应的内容)
	href 	要跳转的内容(路径)
			../Test.html
			访问 上一级目录中的 Test.html
	
	target	决定 打开方式
			默认：_blank	新页面打开
				  _self		本页面打开

锚链接：
	1、设定 命名锚记
	2、创建 锚链接

	命名锚记:
		指要跳转到的位置
		<a name="aa"></a>
		
		name 属性 表示 命名锚记的 名称
	
	锚链接:
		<a href="#aa">跳转到 aa 的锚链接</a>

		#aa 表示 超链接 跳转到 aa 的命名锚记处	
		
		
src 和 href 的区别

目前
src 	属于 <img>	从哪儿来
href	属于 <a>	到哪儿去
	
<ul>	无序列表
<ol>	有序列表
<li>	列表项

<dl>	定义列表
<dt>	相当于标题
<dd>	相当于内容，与 <dt> 间隔 一个制表位


<table>	表格
<tr>	表格的行
<td>	表格的列

大批量展示数据，要求整齐的情况下，
可使用表格来进行局部布局，呈现数据。

<th>	表格的行 或 列 的 标题 标签
		默认 居中 并 加粗
		
<caption>	表格的标题标签
<thead>		表格的页眉标签
<tbody>		表格的主体标签
<tfoot>		表格的页脚标签

框架
<frameset>	框架体
	rows	横向切割
	cols	纵向切割
	
<frame>		填充框架


第三章：

<form>	表单
	action:	提交地址（指定表单元素的值提交到的地方）
	method:	提交方式(get 或 post)
	
<input />	表单元素标签
	type:
			text	文本框(输入明文信息)	
			
			password 密码框(输入暗文信息)	
			
			radio	单选按钮(name 属性必须一致,才是同一组，同一组只能选择一个，圆形选框)
				checked
				checked="checked"
				checked="true"
			
			checkbox复选框(可以选择多个，方形选框)
				checked
			
			submit	提交按钮
				
			reset	重置按钮
			
			button	普通按钮
			
			image	图片按钮(等同于 submit)
			
			file	文件选择控件
			
			
<select>	下拉列表框
<option>	下拉列表的 项
	selected	默认选项
		selected
		selected="selected"
		selected="true"

<textarea>	文本区域
	rows	行数
	cols	列数
	readonly 只读属性
