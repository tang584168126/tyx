编写Java程序的三个步骤：
1、编写源程序 (.java 为后缀的文件)
2、编译源程序，生成字节码文件(.class 为后缀的文件)
3、运行 字节码文件

编译源程序的命令：
javac 

案例：
假设 Java源文件名称为：Test.java

编译：javac Test.java

运行：java Test


JDK扩展知识

假设 JDK 安装目录为：D:\Java\jdk1.6.0_23

1、创建 JAVA_HOME 键
	键值为：D:\Java\jdk1.6.0_23

2、创建 ClassPath 键
	键值为：%JAVA_HOME%\lib;
	
3、在 Path 中 加入
	键值为：     之前的内容;%JAVA_HOME%\bin;

	使用命令：
	
	java -version
	
	测试结果为：
	
C:\>java -version
java version "1.6.0_23"
Java(TM) SE Runtime Environment (build 1.6.0_23-b05)
Java HotSpot(TM) Client VM (build 19.0-b09, mixed mode, sharing)

public static void main(String[] args){
	
}

static：静态的
void：	空的（没有返回值的）
main：	主要的	

String	字符串
[]		数组

args	参数名称

以上为：Java的入口程序，执行Java程序必备的 main()方法

一个名字后面 + () 的 通通称为：方法
