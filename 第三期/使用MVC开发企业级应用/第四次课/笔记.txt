分页的两种方式：
需求：查询弟5页的数据，每页10条数据
1.先排除前面4页的数据，然后在剩下的数据中查询前面10条数据
select top * from Books where CategoryId = 1and Id not in ( select top 10*(5-1) * from Books where CategoryId = 1 order by PublishDate) order by PublishDate
2.排序后为每行从1开始依次给一个编号了，查询41-50行的数据
select * from (select *,Row_Number() over(order by Id) as BookId from Books where CategoryId = 1 order by PublishDate) as Temp where BookId between 40 and 50 order by PublishDate
