数据库操作
	SQL的组成：
		1.DML(数据操作语言)
			insert(新增)、update(修改)、delete(删除)

		2.DCL(数据控制语言)

		3.DQL(数据查询语言)
			select(查询)

		4.DDL(数据定义语言)
			CREATE DATABASE、CREATE TABLE

	SQL的运算符
		1.赋值运算符
			=：把 右边的值 赋予 左边

		2.算术运算符
			+：加
			-：减
			*：乘
			/：除
			%：模

		3.逻辑运算符
			AND：并且
			OR：或者
			NOT：非

		4.关系运算符
			=：等于
			>：大于
			>=：大于等于
			<：小于
			<=：小于等于
			<>：不等于
			!=：不等于(非SQL-92标准)

	SQL的操作
		1.使用数据库
			use 数据库名

		2.新增数据
			语法
				insert into 表名
				(列名1,列名2,...,列名n)
				values
				(值1,值2,...,值n)

			注意事项
				1.列名与值一一对应(数据类型、数据长度、列名与值的个数)
				2.值必须满足列上的约束(主键约束、非空约束、检查约束、外键约束)
				3.对于有引用约束的表，需要先添加主表数据，再添加子表数据
				4.对于有标识列的表，新增数据时，不允许向标识列添加数据

		3.修改数据
			语法
				update 表名 set
				列名1 = 值1,列名2 = 值2,...,列名n = 值n
				where 更新条件

			注意事项
				1.值必须满足列上的约束(数据类型、主键约束、非空约束、检查约束、外键约束)
				2.更新条件可以省略，省略后将修改表中的所有数据
				3.更新条件若不满足，0行受影响

		4.删除数据
			语法
				delete from 表名
				where 删除条件

			注意事项
				1.删除条件可以省略，省略后将删除表中的所有数据
				2.删除条件若不满足，0行受影响
